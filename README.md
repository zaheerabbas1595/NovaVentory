# NovaVentory Landing Page

React and Vite landing page for the NovaVentory Etsy shop.

## Features

- Hero carousel with autoplay product video
- Etsy listing product grid with direct "Buy on Etsy" links
- Responsive layout for desktop and mobile
- Local listing images and product video assets

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
